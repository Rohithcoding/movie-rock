# Using Cinextma

## Quick Start Guide

### Browsing Content
1. **Home Page**
   - Latest releases
   - Trending content
   - Continue watching

2. **Search**
   - Use the search bar in the top navigation
   - Filter by movie or TV show
   - Sort by rating, release date, etc.

3. **Categories**
   - Browse by genre
   - View trending content
   - Explore new releases

### User Account

1. **Registration**
   ```
   - Click "Sign Up" in the top right
   - Enter email and password
   - Verify email address
   ```

2. **Login**
   ```
   - Click "Login"
   - Enter credentials
   - Access your personalized content
   ```

3. **Profile Features**
   - Watch history
   - Bookmarked content
   - Account settings

### Watching Content

1. **Movies**
   - Click on any movie card
   - View details and information
   - Click "Play" to start watching

2. **TV Shows**
   - Select a TV show
   - Choose season and episode
   - Track watched episodes

3. **Player Features**
   - Quality selection
   - Playback speed
   - Auto-play next episode

## Tips & Tricks

1. **Keyboard Shortcuts**
   ```
   Space      - Play/Pause
   F          - Fullscreen
   M          - Mute
   Arrow Keys - Seek forward/backward
   ```

2. **Mobile Features**
   - Swipe gestures
   - Picture-in-Picture
   - Download for offline viewing

3. **Personalization**
   - Theme preferences
   - Language settings
   - Content filters

## Common Issues & Solutions

1. **Playback Issues**
   - Check internet connection
   - Clear browser cache
   - Try different quality settings

2. **Account Access**
   - Reset password
   - Contact support
   - Check email verification

3. **Mobile Optimization**
   - Enable mobile data
   - Update browser
   - Check device compatibility
